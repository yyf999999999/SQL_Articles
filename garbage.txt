        <h2 data-prefix="3">Drizzle ORMの導入と基本操作</h2>
        <div class="second_layer_content">
            <h3 data-prefix="3.1">必要なソフトウェアの準備</h3>
            <p>
                Drizzle ORM を PostgreSQL で使うには、次のソフトウェアが必要です。
            </p>
            <ul>
                <li>Node.js（バージョン 16 以上）</li>
                <li>npm または yarn などのパッケージ管理ツール</li>
                <li>PostgreSQL（ローカルまたは Docker で起動）</li>
                <li>TypeScript を使う場合は tsx や .env などの設定</li>
            </ul>
            <p>
                Drizzle ORMは PostgreSQL との接続に<span class="inline-code">node-postgres</span>（<span class="inline-code">pg</span>）ドライバを使います。npm などでインストールします。
            </p>

            <h3 data-prefix="3.2">プロジェクトの初期化とパッケージのインストール</h3>
            <p>
                まずプロジェクトを新規に作成し、必要なパッケージをインストールします。
            </p>
            <div class="code-card">
                <div class="code-header">
                    <span class="code-title">プロジェクトの初期化（ターミナル）</span>
                    <button class="copy-btn" type="button">コピー</button>
                </div>
                <pre><code class="code-block">mkdir drizzle-postgres-example
cd drizzle-postgres-example
npm init -y</code></pre>
            </div>
            <div class="code-card">
                <div class="code-header">
                    <span class="code-title">Drizzle ORM と関連パッケージのインストール</span>
                    <button class="copy-btn" type="button">コピー</button>
                </div>
                <pre><code class="code-block">npm install drizzle-orm pg dotenv
npm install -D drizzle-kit tsx @types/pg</code></pre>
            </div>
            <p class="note-small">
                ※ インストール時に deprecated の警告が出ることがありますが、これは Drizzle Kit の内部で古いパッケージに依存しているためです。<br>
                Drizzle ORM や PostgreSQL 接続自体には影響しないため、警告を無視して進めて問題ありません。
            </p>
            <p>
                ここでは次のパッケージを導入しています：
            </p>
            <ul>
                <li><span class="inline-code">drizzle-orm</span>: Drizzle ORM 本体</li>
                <li><span class="inline-code">pg</span>: PostgreSQL 用 Node.js ドライバ</li>
                <li><span class="inline-code">dotenv</span>: <span class="inline-code">.env</span>ファイルから環境変数を読み込むため</li>
                <li><span class="inline-code">drizzle-kit</span>: Drizzle ORM 公式のツール（スキーマ管理・マイグレーション用）</li>
                <li><span class="inline-code">tsx</span>: TypeScript を簡単に実行するツール</li>
                <li><span class="inline-code">@types/pg</span>: <span class="inline-code">pg</span>ドライバ用の型定義（TypeScript 用）</li>
            </ul>

            <h3 data-prefix="3.3">PostgreSQL 接続情報の設定</h3>
            <p>
                プロジェクトルートに<span class="inline-code">.env</span>ファイルを作成し、PostgreSQL 接続情報を記述します。例として以下のようにします：
            </p>
            <div class="code-card">
                <div class="code-header">
                    <span class="code-title">PostgreSQL 接続情報</span>
                    <button class="copy-btn" type="button">コピー</button>
                </div>
                <pre><code class="code-block">DATABASE_URL=postgresql://postgres:password@localhost:5432/mydb</code></pre>
            </div>
            <ul>
                <li><span class="inline-code">postgresql://</span>: PostgreSQL の接続プロトコル</li>
                <li><span class="inline-code">postgres</span>: DB のユーザ名</li>
                <li><span class="inline-code">password</span>: パスワード</li>
                <li><span class="inline-code">localhost</span>: DB ホスト名</li>
                <li><span class="inline-code">5432</span>: PostgreSQL のデフォルトポート</li>
                <li><span class="inline-code">mydb</span>: 接続したいデータベース名</li>
            </ul>
            <p>
                環境変数を使うことで、コードに直接接続情報を書かずセキュアに管理できます。
            </p>

            <h3 data-prefix="3.4">Drizzle ORM の準備</h3>
            <p>
                プロジェクトの<span class="inline-code">src</span>フォルダに<span class="inline-code">index.ts</span>として接続コードを書きます。ここでは TypeScript を前提としたコード例を示します。
            </p>
            <div class="code-card">
                <div class="code-header">
                    <span class="code-title">Drizzle ORM PostgreSQL 接続コード</span>
                    <button class="copy-btn" type="button">コピー</button>
                </div>
                <pre><code class="code-block">import 'dotenv/config';
import { drizzle } from 'drizzle-orm/node-postgres';
import { Pool } from 'pg';

// PostgreSQL の接続プールを作成
const pool = new Pool({
  connectionString: process.env.DATABASE_URL,
});

// Drizzle ORM を初期化
export const db = drizzle(pool);

// 簡単なクエリ例
async function testConnection() {
  const result = await db.execute('SELECT 1');
  console.log('Connection test result:', result);
}

testConnection();</code></pre>
            </div>
            <p>
                このコードでは：
            </p>
            <ul>
                <li><span class="inline-code">.env</span>ファイルから環境変数を読み込んでいます</li>
                <li><span class="inline-code">Pool</span>で接続プールを作成</li>
                <li><span class="inline-code">drizzle()</span>で Drizzle ORM のインスタンスを生成しています</li>
                <li><span class="inline-code">db.execute('SELECT 1')</span>で簡単な接続確認を行っています</li>
            </ul>

            <h3 data-prefix="3.5">PostgreSQL への接続確認</h3>
            <p>
                ターミナルで次のように実行します：
            </p>
            <div class="code-card">
                <div class="code-header">
                    <span class="code-title">接続確認コマンド</span>
                    <button class="copy-btn" type="button">コピー</button>
                </div>
                <pre><code class="code-block">npx tsx src/index.ts</code></pre>

            <p>
                下記のように<span class="inline-code">SELECT 1</span>の結果が表示されれば、Drizzle ORM が PostgreSQL に接続できています。

            <div class="code-card">
